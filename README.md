# 👾 Centipede 2D Game

A retro 2D arcade game built with **Python** & **Pygame**, supporting desktop platforms and Android deployment via **Buildozer**.

![Pygame](https://img.shields.io/badge/Pygame-ED1C24?style=for-the-badge&logo=python&logoColor=white)
![Android](https://img.shields.io/badge/Android_Buildozer-3DDC84?style=for-the-badge&logo=android&logoColor=white)

---

## 🌟 Features

- 🐛 **Centipede Arcade Gameplay**: Defend against moving centipedes, obstacles, and enemy sprites.
- 🎨 **Custom Pixel Art & Fonts**: Original retro sprite sheets and micro pixel typography.
- 🔊 **Sound & FX**: Retro sound effects, collision detection, and score tracking.
- 📱 **Multi-Platform Support**: Playable on Desktop (Windows/Mac/Linux) and mobile packaging via Buildozer (`buildozer.spec`).

---

## 🚀 How to Run

### Desktop (Python)
```bash
git clone https://github.com/AkBePEEK/Game_Project.git
cd Game_Project
pip install pygame
python main.py
```
